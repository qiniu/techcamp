# 推动 LLGo 对 Python 库开箱即用能力的建设

## 目标

让 LLGo 用户能够通过简单命令（如 `llgo get numpy`）自动安装和使用 Python 库，无需手动配置环境，实现跨平台一致的依赖管理体验。
